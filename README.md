# ProctoLearn — Онлайн оқу платформасы және прокторинг жүйесі (Kazakh)

ProctoLearn — бұл онлайн оқу платформасы: курстар, сабақтар, тесттер/емтихандар және емтихан кезінде прокторинг (анти-списывание) функционалы бар жүйе.  
Жоба интерфейсі толық **қазақ тілінде** жоспарланған.

## 1) Негізгі мүмкіндіктер

### 1.1 Курстар
- Курстар каталогы
- Сабақтар (видео/материалдар)
- Прогресс (оқу барысы)

### 1.2 Емтихан
- Серверлік таймер (уақытты қолдан өзгертуге қиын)
- Авто-сақтау (жауаптарды периодты сақтау)
- Авто-тапсыру (уақыт біткенде)

### 1.3 Прокторинг (анти-списывание)
- Камераны міндетті қосу (емтиханды бастау шарты)
- Экран бөлісу (screen share) міндетті/қосымша
- Оқиғаларды тіркеу:
  - қойынды ауыстыру (tab switch)
  - fullscreen режимінен шығу
  - copy/paste әрекеттері
- Evidence (дәлел) сақтау:
  - скриншоттар
  - қысқа клиптер (қалауы бойынша)
- Сенім ұпайы (Trust Score) — күмәнді әрекеттер негізінде есептеу

### 1.4 Әкімші/Проктор панелі
- Емтихандарды реалтайм бақылау (Socket.io)
- Әр әрекеттің таймлайны
- Evidence қарау
- Шешім: «Қабылдау / Қайта тапсыру / Жарамсыз»

---

## 2) Технологиялық стек

**Frontend:** Next.js (React)  
**Backend:** NestJS (Node.js)  
**DB:** PostgreSQL  
**Realtime:** Socket.io  
**Storage:** MinIO (S3 үйлесімді)  
**Deploy:** Docker + Docker Compose
