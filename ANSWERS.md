# Барлық емтихан сұрақтарының жауаптары

## Курс 1 — JavaScript негіздері
1. JavaScript-те айнымалы жариялаудың дұрыс жолы? → **var x = 5**
2. typeof null нені қайтарады? → **object**
3. Массив ұзындығын алатын қасиет? → **.length**
4. DOM-дан элементті алу үшін? → **getElementById**
5. async/await не үшін қолданылады? → **Асинхронды операцияларды күту**

## Курс 2 — Python бағдарламалау
1. Python-да тізімді жариялау? → **x = [1,2,3]**
2. Python-да функция кілт сөзі? → **def**
3. Сөздіктегі кілттерді алу? → **.keys()**
4. Python-да файлды оқу? → **open('f','r')**
5. range(5) нені береді? → **[0,1,2,3,4]**

## Курс 3 — SQL және реляциялық мәліметтер қоры

2. Жаңа жазба қосатын команда? → **INSERT INTO**
3. Екі кестені біріктіру? → **JOIN**
4. Жылдам іздеу үшін? → **INDEX**
5. Транзакцияны растау? → **COMMIT**

## Курс 7 — Деректер құрылымдары мен алгоритмдер
1. LIFO принципімен жұмыс істейтін құрылым? → **Стек**
2. FIFO принципімен жұмыс істейтін құрылым? → **Кезек**
3. Binary Search күрделілігі? → **O(log n)**
4. Бинарлық ағаштың жоғарғы элементі? → **Root**
5. Bubble Sort күрделілігі? → **O(n^2)**

## Курс 10 — Кибер қауіпсіздік негіздері
1. SQL инъекциясынан қорғаудың тиімді әдісі? → **Параметрленген сұраныстар**
2. XSS шабуылы нені мақсат етеді? → **Пайдаланушы браузері**
3. HTTPS пен HTTP айырмашылығы? → **Шифрлау**
4. TLS не үшін қолданылады? → **Шифрлауда**
5. Brute Force-тан қорғану? → **Rate limiting және CAPTCHA**
